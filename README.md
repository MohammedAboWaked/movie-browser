# movie-browser
Movie browsing app using HTML, CSS, and JavaScript with API integration.

Movie Browsing App

📖 Overview

This is a movie browsing web application built using HTML, CSS, and Vanilla JavaScript.
The app fetches movie data from an external API and displays it in an interactive UI.

How to Run the Project

Download or clone the repository

Open the project folder

Open index.html in your browser

Make sure you have an internet connection (for API requests)

📌 Assumptions
I assumed that all movies have a poster image, a movie title, and a duration.

⚠️ Limitations

Some API responses may have incomplete data
